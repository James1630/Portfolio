School Project

E-commerce

This project is an online shoe store built using Laravel, a PHP framework, designed to provide a smooth and interactive shopping experience. Users can browse through different shoe categories, view product details, add items to their shopping cart, and place orders securely.

The system includes user authentication, allowing customers to register, log in, and manage their profiles. Admins have a separate dashboard where they can manage products, update stock, view orders, and track sales.

Features include :

Product Management: Admins can add, edit, or remove products with images, prices, and descriptions.

Cart & Checkout : Users can add multiple items to their cart, update quantities, and proceed to a checkout page.

Order Management: Admins and users can track orders and view order history.

Responsive Design: The site is mobile-friendly and works on desktops, tablets, and smartphones.

Database Integration: All data (users, products, orders) is stored securely using MySQL, with Laravel handling backend operations efficiently.

This project demonstrates practical knowledge of web development, database management, and e-commerce workflow, providing a complete end-to-end solution for online retail.

Hijo Resource Corporation OJT Project

Ticketing System

This project involves designing and implementing a comprehensive ticketing system leveraging Microsoft Power Platform tools, including SharePoint, Power Apps, Power Automate, and Power BI. The goal is to streamline the management of service requests, automate workflows, and provide real-time insights for improved decision-making.

• SharePoint is utilized as the central data repository, storing ticket details such as requestor information, issue type, priority, status, and assigned agents. Its robust document management capabilities ensure all tickets and related documents are organized and easily accessible.

•Power Apps is employed to develop a user-friendly interface for different stakeholders. Clients can submit tickets, while Admins and Super Admins can track, update, and assign tickets seamlessly through intuitive forms and dashboards.

•Power Automate enables workflow automation, including ticket assignment, notifications, escalation procedures, and status updates. This reduces manual effort, ensures timely responses, and improves overall efficiency.

•Power BI provides visual analytics and reporting, offering real-time insights into ticket trends, agent performance, and service efficiency. Interactive dashboards help management make informed decisions and identify areas for process improvement.

By integrating these Power Platform tools, the ticketing system transforms manual and fragmented processes into a centralized, automated, and data-driven solution, enhancing productivity, transparency, and service quality across the organization.
